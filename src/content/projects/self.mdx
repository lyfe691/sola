# Bringing Neofetch to Windows

**Self** is a customizable Windows system information display tool that brings the beloved Unix/Linux Neofetch experience to Windows users. Built entirely with Python, it provides a clean, customizable way to display system information alongside ASCII art or images.

> "Why should Linux users have all the fun?" - The motivation behind Self

The goal was to create a native Windows solution that feels familiar to Neofetch users while leveraging PowerShell's capabilities.

## Technical Architecture

Self is built with **Python** as the core language, utilizing Windows APIs and system calls for accurate information gathering.

## Usage & Configuration

### Command Line Interface

Self provides a comprehensive CLI with multiple options:

<CodeBlock
  code={`# Basic usage
self                 # Run with default settings
self --setup         # Interactive setup wizard
self --update        # Update to latest version

# Customization options

self --image <file> # Use specific image
self --ascii <file> # Use ASCII art instead
self --height <n> # Set image height
self --width <n> # Set image width

# Advanced options

self --config <path> # Use custom config file
self --no-cache # Skip cached system info
self --debug # Run in simplified debug mode
self --version # Show version information`}
